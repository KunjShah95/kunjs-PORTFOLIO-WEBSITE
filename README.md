# Kunj Shah — Portfolio

Personal portfolio of Kunj Shah, an AI engineer and agent builder based in Ahmedabad, shipping production-ready autonomous AI agents, full-stack web apps, and APIs.

Built with React, TypeScript, Tailwind CSS, Framer Motion, and Vite. Deployed on Vercel + Cloudflare.

## Stack

- **Framework:** React 18 + TypeScript
- **Build:** Vite 8
- **Styling:** Tailwind CSS 3
- **Animation:** Framer Motion
- **Deployment:** Vercel (frontend + API routes), Cloudflare Workers (prerendering)
- **Other:** react-router-dom, lucide-react, react-helmet-async

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start dev server |
| `npm run build` | Type-check + build + prerender routes |
| `npm run typecheck` | TypeScript type-checking |
| `npm run preview` | Build + preview via wrangler |
| `npm run deploy` | Build + deploy to Cloudflare |
